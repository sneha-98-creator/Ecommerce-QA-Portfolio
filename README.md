## 📌 Project Overview

**ShopEasy** is a fictional enterprise-level E-Commerce application developed to demonstrate end-to-end Software Testing activities followed in Agile software development.

This repository showcases complete QA documentation including requirement analysis, test planning, test case design, execution, defect reporting, traceability, and test summary reporting.

---

## 🎯 Project Objectives

- Validate all business-critical workflows.
- Ensure application quality before production deployment.
- Identify and report defects effectively.
- Demonstrate professional QA documentation practices.
