# Project Overview

## Project Name

**ShopEasy - E-Commerce Platform**

---

## Domain

Retail / E-Commerce

---

## Project Description

ShopEasy is a cloud-based E-Commerce platform that allows customers to browse products, add items to the shopping cart, securely complete purchases, and manage their orders online.

The application is designed to provide a seamless shopping experience across desktop and mobile devices.

---

## Business Objective

The primary objective of ShopEasy is to provide customers with a secure, reliable, and user-friendly online shopping platform while ensuring high software quality through comprehensive testing.

---

## Application Modules

- Registration

- Shopping Cart
